# Neuroflow SaaS

Web-first SaaS concept for ADHD / neurodivergent users.

## Working Direction

This product is not trying to be "another todo app."

It is meant to reduce friction, lower overwhelm, and make progress visible in a way that feels encouraging instead of punishing.

## MVP Positioning

- visual progress loops
- low-friction task capture
- gentle daily reset
- neurodivergent-friendly structure
- habit tracking without shame

## First Build Goal

Build a lightweight web app where a user can:

1. create a small set of focus areas
2. tick progress visually
3. see streak-like momentum without aggressive pressure
4. reset or recover after a missed day without feeling "behind"

## Suggested Free Stack

- VS Code for local work
- GitHub for source control
- Vercel for deployment
- Supabase free tier for auth + database later
- Stripe later for payments/trial logic
- n8n for reminders, onboarding flows, and check-in automations

## Honest Build Advice

Do not start with Replit as the center of the product if you already have VS Code, GitHub, and Vercel.

Better:

- build in VS Code
- push to GitHub
- deploy to Vercel
- use Replit only if you need a quick sandbox

## Subscription Advice

Yes, a one-month free trial is sensible.

For MVP:

- free public landing page
- free account creation later
- 30-day free trial
- paid plan only after people actually use the product

Do not overbuild billing first.
