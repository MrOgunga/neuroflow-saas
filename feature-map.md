# Suggested Feature Topics

## Best MVP Topics To Start With

These are the best starting modules for this kind of product:

1. Habit / progress tiles
2. Focus session tracker
3. Gentle daily planning
4. Recovery mode after missed days
5. Dopamine-safe rewards / wins

## Strong Habit Tile Ideas

- reading
- running
- hydration
- no junk food
- meds taken
- inbox cleared
- deep work
- sleep before midnight
- journal
- stretch

## ADHD / Neurodivergent-Friendly Angles

- "start tiny"
- "done is visible"
- "missed day is not failure"
- "your brain is not broken"
- "friction is the enemy"
