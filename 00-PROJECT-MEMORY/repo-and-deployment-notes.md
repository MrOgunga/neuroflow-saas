# Repo and Deployment Notes

## Repo Visibility

Current GitHub repo visibility:

- `PUBLIC`

Repo:

- `https://github.com/MrOgunga/neuroflow-saas`

## Hosting Model

Current deployment flow:

1. build/edit locally in VS Code
2. push to GitHub
3. Vercel deploys the web app

## Replit

Replit is optional.

It can be used as:

- a sandbox
- a quick prototype environment

But it is not the main source of truth for this project.

## Main Source of Truth

The real project should live in:

- local project folder
- GitHub repo
- Vercel deployment

## Recommendation

If you want the repo private, switch it to private before adding real production keys.
