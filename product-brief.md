# Product Brief

## Working Name

Neuroflow

This is a working name, not a final brand decision.

## Core Problem

A lot of ADHD / neurodivergent users do not struggle because they "do not care."

They struggle because many tools are:

- too cluttered
- too rigid
- too guilt-heavy
- too text-dense
- built around punishment instead of recovery

## Core Promise

"A calmer progress system for brains that do not work in straight lines."

## MVP User

Start with one very specific user:

- creative adults with ADHD traits
- founders / freelancers / students
- people who need visual momentum and light structure

## MVP Features

1. Focus tiles
   - reading
   - running
   - no junk food
   - deep work
   - sleep

2. Visual tap-to-complete grid
   - one tap marks progress
   - each habit shows a satisfying pattern

3. Daily recovery framing
   - missed days do not destroy the experience
   - copy should encourage return, not shame

4. Simple progress stats
   - consistency
   - current run
   - this week total

5. Gentle dashboard
   - "what can I do next?"
   - no heavy admin feeling

## What Makes It Different

- visual-first instead of spreadsheet-first
- soft recovery model instead of streak obsession
- lower cognitive load than Notion/OneNote setups
- built for real inconsistency, not perfect discipline

## SaaS Direction

### Free Version

- 3 focus tiles
- basic visual tracker
- one device/browser

### Paid Version

- unlimited focus tiles
- routines
- reminders
- reflections
- templates
- export/history
- adaptive weekly reset

## First Revenue Logic

- 30-day free trial
- monthly subscription after that
- maybe annual discount later

## What Not To Build First

- full social features
- AI coaching
- complex analytics
- too many settings
- multiple personas

## MVP Rule

Make it beautiful, calming, and easy enough that someone can use it in under 30 seconds.
